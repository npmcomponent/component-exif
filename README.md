# exif

  Component fork of [mattiasw/ExifReader](https://github.com/mattiasw/ExifReader) with a simplified API.

## Installation

```
$ component install component/exif
```

## API

### exif(arraybuffer)

  Return an object of tags from the given `arraybuffer`.
